# Examensarbete – Extraktion av strokekohorter från MIMIC-IV med algoritmiska metoder
Detta projekt utgör den praktiska implementationen av examensarbetet
"Utveckling och utvärdering av algoritmer för att extrahera kliniska studiekohorter från verkliga intensivvårdsdata: en fallstudie på stroke med MIMIC-IV"

Projektet syftar till att utveckla, implementera och utvärdera tre algoritmiska angreppssätt för automatisk kohortgenerering från MIMIC-IV, med särskilt fokus på strokepatienter som vårdats på intensivvårdsavdelning.
